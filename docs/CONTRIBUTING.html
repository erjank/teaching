Contributing

  1. Read the GitBook documentation.
  2. Create a branch in your desktop Git repository.
  3. Edit the Markdown files in this directory and its subdirectories.
  4. Commit your changes.
  5. Push that branch to your repository on GitHub.
  6. Submit a pull request from your branch in your repository to the master repository.
